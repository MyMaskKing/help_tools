# 一键构建Dockerfile并启动
`docker compose -f cloudflare-compose.yaml up --build --force-recreate -d`
# 分步1.构建Dockerfile
docker compose -f cloudflare-compose.yaml build --no-cache

- 遇到无法下载alipine的请看，请手动执行`docker pull alpine:latest`

# 分步2. 启动服务
docker compose -f cloudflare-compose.yaml up -d
- 强制重新部署
docker compose -f cloudflare-compose.yaml up --force-recreate -d

# 注意内容
## 1.更新镜像请执行下面命令
> 由于Dockerfile在构建的时候默认去缓存中查找，所以需要手动更新最新的cloudflared镜像
- 拉取最新的 Alpine 基础镜像
docker pull alpine:latest

- 拉取最新的 Cloudflared 可执行文件源镜像
docker pull cloudflare/cloudflared:latest
